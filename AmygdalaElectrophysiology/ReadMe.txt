The code shows neutral and scary images, and the let the user rate them from 1 (neutral) to 3 (scary).
If the rating is 3 - it will send Microbit 'S', 15 times, in 100 ms intervals.
Otherwise it will send 'C' in the same timing.
 
The code on the Microbit will respond according to the codeForMicrobit.hex file (stormy response to S, calm response to C)
